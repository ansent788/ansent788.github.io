### 简介

MediaPipe 以预构建的 Python 包的形式提供即用型但可自定义的 Python 解决方案。 MediaPipe Python 包适用于 Linux、macOS 和 Windows。

<!-- more -->

### 安装

```bash
pip install mediapipe
```
